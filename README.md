# Twitter Stock Market Data Cleaning & Visualization

## Project Overview
This project focuses on cleaning and analyzing Twitter stock market data. The dataset is processed to handle missing values and correct data types. After cleaning, the project generates visualizations using Matplotlib, including:

- Closing price trends over time  
- High-Low price ranges  
 
All cleaned data and chart images are saved in an organized `output/` folder for easy analysis and reporting.

## Features
- Data cleaning: handle missing values and correct data types  
- Data transformation: calculate high-low ranges and closing price trends 
- Visualization: line plots   
- Outputs saved as CSV and chart images for reuse
